# Personal Portfolio Website

## Description
Personal portfolio website demonstrating frontend skills through interactive examples.
Includes HTML, CSS, and JavaScript demos, and a contact form.

## Goals
- Showcase real frontend skills
- Provide live demonstrations instead of descriptions
- Allow recruiters to contact me easily

## Target Audience
- HR specialists
- Frontend / Team Leads

## Features
- Responsive layout
- Interactive skill demos
- Clean semantic HTML
- Contact form

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
